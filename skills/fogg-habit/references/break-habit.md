# 戒除坏习惯模式：反向行为设计

用于用户想**改掉/停掉**一个习惯（刷手机、熬夜、奶茶、咬指甲……）。福格的主张：戒除坏习惯不是靠忍，而是**反向操作 B=MAP**——拆掉提示、加大难度、削弱动机，再用一个好习惯顶替它的位置。

福格的总思路：**从笼统到具体（General → Specific）**。"坏习惯"往往是一团，先拆成一个个具体行为，再逐个击破。

## 1. 把"坏习惯"拆具体

先别笼统说"我要少玩手机"。问用户：
- "具体是哪个动作、在什么场景下发生？"（"晚上躺床上刷短视频刷到一两点"）
- "它通常被什么触发？"（无聊？焦虑？某个固定时间/地点？）

把它还原成一个**具体行为 + 它的提示 + 它给你的即时回报**。拆得越细越好打。

## 2. 三条破除路径（反向 B=MAP）

行为 = 动机 × 能力 × 提示。要让一个行为**不发生**，反着拆任意一环，能同时拆多环更好：

| 拆哪一环 | 怎么做 | 例子 |
|---|---|---|
| **移除提示（最有效）** | 找到触发这个坏习惯的提示，把它拿掉/挡住 | 手机不进卧室；卸载 App；把零食搬出视线 |
| **加大难度（降低能力）** | 给行为增加时间/体力/步骤上的摩擦 | App 退出登录每次要重输密码；奶茶要走 20 分钟才买得到 |
| **削弱动机** | 减少它的即时快感，或放大它的真实代价 | 把熬夜的代价具体化、可见化；取关让你上瘾的内容源 |

福格的经验：**移除提示通常最干净有效**，因为没有提示，行为根本不会被触发，比靠意志力硬抗省力得多。

## 3. 优先用"顶替"而不是"硬戒"

直接删掉一个习惯会留下空缺，容易反弹。更稳的做法是**在同一个触发点上，种一个新的好习惯顶上去**：
- 用同一个锚点，把坏行为换成一个微小的好行为。
- 例："一坐到沙发上就摸手机" → 改成"一坐到沙发上，就拿起旁边放好的书读一句"。
- 这其实就是回到**行为设计模式**（`design.md`）：给替代行为做微行为 + 锚点 + 庆祝。

## 4. 从最容易撼动的那根线下手

不必一次拆光三环。和用户一起判断：**移除提示、加难度、换行为，哪一个对他来说当下最好实施**，先动那一个，见效了再加码。福格强调**先赢一把**，正向情绪会带动后面。

同样别让用户自责——戒不掉不是不够自律，是某一环的设计还没拆到位。

## 5. 收尾

- 给用户一个清楚的方案：**拆哪个提示 + 加什么摩擦 + （可选）用什么好习惯顶替**。
- 预告：观察一两周，哪根线又松回去了，回来调。
- 别忘 SKILL.md 结尾：问要不要把这个方案整理成一份小结。
